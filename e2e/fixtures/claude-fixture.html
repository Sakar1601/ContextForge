How do I set up a pnpm monorepo?
Create a pnpm-workspace.yaml file at the root listing your package globs, then add a root package.json with workspaces pointing to the same paths.
Should I use TypeScript strict mode?
Yes, strict mode catches many common bugs at compile time. Enable it via "strict": true in your tsconfig.json compilerOptions.
